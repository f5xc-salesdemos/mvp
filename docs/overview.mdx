## Purpose

This document presents the research, design, and delivery plan for **mvp**, a capability program that amplifies F5 Distributed Cloud practitioners by offloading repetitive knowledge-gathering and artifact-production work to an agentic subject matter expert. The aim is to free humans to focus on the high-leverage work only they can do — judgment, relationships, interpretation, and decisions.

We borrow the *minimum viable product* methodology to guide *how* we build the capability: iterate fast, prove value early, expand scope deliberately. The word "product" is methodological; xcsh is not being sold. **xcsh is the tool** we build to deliver the capability — an agentic shell purpose-built to encode F5 Distributed Cloud expertise and automate platform operations on behalf of the human.

## Naming

- **mvp** is the program — the capability we deliver to F5 Distributed Cloud practitioners. Always lowercase; never MVP, Mvp, or mVp.
- **xcsh** is the tool we build to deliver mvp. Always lowercase; never Xcsh, XCSh, or XCSH.
